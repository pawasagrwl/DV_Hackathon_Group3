# DV Hackathon Group 3

This repository contains the data files and Tableau workbook used for our Data Visualization Hackathon project.

## Contents

### Data Files
- **station-hourly.csv** (3.0 MB) - Hourly station-level ridership data
- **stationpair-hourly-enhanced.csv** (76 MB) - Enhanced hourly station pair ridership data

### Tableau Workbook
- **Book1.twbx** (12 MB) - Tableau workbook containing our visualizations and analysis

## Usage

To view the visualizations, open `Book1.twbx` in Tableau Desktop or Tableau Reader.
